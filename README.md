# hjamaan.github.io

Personal academic website for Dr. Hamoud Aljamaan, hosted on GitHub Pages.

## Local preview

```bash
python3 -m http.server 8000
```

Open http://localhost:8000 in your browser.

## Deploy

Push to the `main` branch. GitHub Pages serves static files from the repository root.

Site URL: https://hjamaan.github.io/
